    Enum SettingsKeyType

    Key type for settings entry

    Namespace: Nitrocid.Kernel.Configuration.Settings
    Assembly: Nitrocid.dll
    Syntax
    public enum SettingsKeyType

    Fields

    Name Description
    SBoolean

    The value is of bool
    SChar

    The value is of char and only accepts one character.
    SColor

    The value is of Terminaux.Colors.Color and comes from the color wheel
    SDouble

    The value is of double
    SFiglet

    The value is a Figlet font name that allows Settings to use the Figlet font selector.
    SIcon

    The value is an icon font name that allows Settings to use the icon selector.
    SInt

    The value is of int
    SIntSlider

    The value is of int, but has a slider which has a minimum and maximum value. Useful for numbers which are limited.
    SList

    The value is of System.Collections.Generic.IEnumerable<T>, like System.Collections.Generic.List<T>
    SMultivar

    The value consists of multiple variables.
    SPreset

    The value is a shell preset defined using IPromptPreset in Nitrocid.Shell.Prompts.PromptPresetManager.CurrentPresets
    SSelection

    The value is of the selection, which can either come from enums, or from System.Collections.Generic.IEnumerable<T>, like System.Collections.Generic.List<T>
    SString

    The value is of string
    SUnknown

    Unknown type
    SVariant

    The value consists of variant function that parses the input and converts it to its appropriate config value
