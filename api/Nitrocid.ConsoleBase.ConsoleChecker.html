    CheckConsole()

    Checks the running console for sanity, like the incompatible consoles, insane console types, etc.
    The severity of the checks can be described in two categories:

    1. Error: We'll throw KernelException with type InsaneConsoleDetected when we detect that the console is not conforming to the standards.
    2. Warning: We'll just issue a warning in yellow text using the plain 16-color writer when we detect that the console is not supporting optional features, like 256-colors.
