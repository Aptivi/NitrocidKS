Conmemorando el 5º aniversario del kernel
Primer lanzamiento
Este kernel se lanzó por primera vez el 22 de febrero de 2018 para crear un simulador minimalista que realmente simula cómo funciona el núcleo del kernel.
Solía albergar solo un shell y algunos comandos, y solo funcionaba con Windows debido al uso de la herramienta Instrumental de administración de Windows.
Tampoco contaba con documentación propia.
Refinamiento del núcleo
Se lanzaron versiones del kernel, como la segunda versión principal para agregar colores y características básicas, y la cuarta versión principal para agregar configuración.
La documentación comenzó en la sexta versión principal y poco a poco se han agregado varias cosas interesantes al kernel.
Versión actual
Esta versión ahora refina el kernel hasta el punto de que ya no se comporta como las versiones anteriores.
Debido a que se lanzaron nuevas características innovadoras, decidimos nombrar este kernel...
